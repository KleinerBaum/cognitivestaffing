# Onboarding source inputs preview / Vorschau der Onboarding-Quellen

Binary screenshots are unavailable in this environment, so the onboarding URL vs. upload layout is documented as text. The layout below mirrors the two-panel OR structure.

```
┌───────────────────────────── URL ─────────────────────────────┐   ┌──────────────────────────── Upload ───────────────────────────┐
│ Stellenanzeigen-URL hinzufügen                                 │   │ Stellenanzeige hochladen (PDF/DOCX/TXT)                        │
│ [ Text input ]                                                 │   │ [ File uploader ]                                              │
│ Für öffentliche Karriereseiten oder Jobbörsen.                 │   │ Ideal für interne Dokumente oder passwortgeschützte Dateien.   │
└───────────────────────────────────────────────────────────────┘   └──────────────────────────────────────────────────────────────┘
                              ───────── OR/ODER ─────────
```

Use this preview to verify the layout while the binary screenshots are generated separately.
